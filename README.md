# Debris Flow–Transmission Tower Impact Dataset

This dataset contains 1,120 high-fidelity SPH–FEM simulation results 
of debris flow impact on a transmission tower.

## Variables
| Column | Description | Unit |
|--------|-------------|------|
| volume_m3 | Debris flow volume (Vdf) | m³ |
| slope_deg | Slope angle (θ) | ° |
| potential_energy_MJ | Debris flow potential energy (PEdf) | MJ |
| plastic_energy_MJ_front | Front leg plastic energy (Ef) | MJ |
| plastic_energy_MJ_rear | Rear leg plastic energy (Er) | MJ |
| sleout_energy_max_J | Impact energy (Eimp) | J |
| impact_duration_s | Contact duration (tc) | s |
| rcforce_force_max_N | Maximum impact force (Fmax) | N |
| rcforce_impulse_Ns | Impulse (I) | N·s |
